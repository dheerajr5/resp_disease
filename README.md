# Respiratory disease shiny app 

### This app is about the air pollution metric PM 2.5 and death rate due to respiratory diseases all over world. Make use of shinydashboard and shinywidgets for UI. 

### Dataset Info
 - [World bank](https://data.worldbank.org/indicator/EN.ATM.PM25.MC.M3)
 - [OCED](https://stats.oecd.org/Index.aspx?DataSetCode=EXP_PM2_5)
 - [Respiratory death rate over world](https://ourworldindata.org/grapher/respiratory-disease-death-rate)
